date: 2026-03-03 00:00:00 +0000
---

This repository now includes a GitHub Pages-powered blog plus a centralized **Try Subprojects** page.

To add a new runnable subproject entry:

1. Create a markdown file under `docs/_subprojects/`.
2. Set `title`, `summary`, `repo_path`, and `try_url` in front matter.
3. Publish the subproject (for example in a folder served by GitHub Pages).

Once merged, the entry will appear automatically at `/try/`.
